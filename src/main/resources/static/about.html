About This Project

A simple full-stack CRUD gallery: Spring Boot + JPA for the API and Bootstrap + fetch() on the client. Data is persisted to your database and rendered from the browser.

  • View all animals
  • Create, edit, and delete entries
  • Responsive, accessible, dark UI
